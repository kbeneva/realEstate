## Projet - Real estate | HomeBreeze

---
### Décription du projet

L’application « HomeBreeze » est une plateforme où les utilisateurs peuvent rechercher des propriétés dans une province ou un quartier spécifique.
Grâce à des filtres personnalisés (comme le prix, le nombre de chambres, etc.), ils peuvent raffiner leur recherche pour trouver la propriété idéale.
HomeBreeze offre la possibilité de gérer des offres immobilières, aussi bien pour les acheteurs que pour les vendeurs.
